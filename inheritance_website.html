Inheritance in Java

Introduction to Inheritance in Java

Inheritance is a fundamental concept in object-oriented programming (OOP). It allows a class to inherit the properties and behaviors of another class.

Java supports single inheritance, where a class can inherit from only one superclass, and multiple inheritance through interfaces.

Example:

class Animal {
    void eat() {
        System.out.println("Eating...");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Barking...");
    }
}

In this example, the Dog class inherits from the Animal class, and it has its own behavior (bark()) in addition to the inherited behavior.
